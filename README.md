## 💾 Requisitos do Sistema

### Recomendações Mínimas

- **RAM**: 12 - 16 GB alocada
- **Processador**: Intel i7/Ryzen 5 ou superior
- **Armazenamento**: Mínimo 20 GB de espaço livre
- **Java**: 17+ (compatível com Neo Forge)

> ⚠️ Devido à quantidade de mods, configurações com menos de 12 GB podem apresentar lag

## 🚀 Instalação

### 1. Pré-requisitos

- [Minecraft Launcher](https://launcher.mojang.com/) instalado
- [Neo Forge 1.21.1](https://neoforged.net/) instalador

### 2. Passos de Instalação

1. Clone ou baixe este repositório
   ```bash
   git clone https://github.com/rafamendes9/God-of-Create.git
   ```

2. Navegue até a pasta `God of Create`

3. Copie todos os arquivos de configuração para sua pasta `.minecraft`
   - Arquivos em `configureddefaults/` serão sincronizados automaticamente

4. Instale Neo Forge 1.21.1

5. Inicie o Minecraft e selecione o perfil Neo Forge

6. Na primeira inicialização, os mods serão baixados e configurados automaticamente

### 3. Aplicar Seed Recomendada

Ao criar novo mundo:
- **Seed**: `3866829312109018270`
- **Centro Spawn**: X: -6000 | Z: 321

## 📁 Estrutura do Projeto

```
God-of-Create/
├── God of Create/
│   ├── config/              # Configurações de mods
│   ├── configureddefaults/  # Defaults sincronizados com .minecraft
│   ├── mods/                # Mods instalados
│   └── options.txt          # Opções do cliente
├── .gitattributes           # Configurações Git
└── README.md                # Este arquivo
```

### 📝 Sistema de Configuração (configureddefaults)

O diretório `configureddefaults/` funciona como espelho sincronizado do `.minecraft`:
- ✅ Arquivos são copiados automaticamente na inicialização
- ✅ Apenas copia se o destino estiver vazio (não sobrescreve)
- ✅ Perfeito para compartilhar configurações entre jogadores

## 🔧 Customizações

### Modificar Configurações

1. Edite os arquivos em `config/`
2. Altere `options.txt` conforme necessário
3. Aplique o arquivo `crash_assistant_localization_overrides/` se precisar customizar idiomas

### Adicionar Novos Mods

1. Baixe o arquivo `.jar` do mod compatível com Neo Forge 1.21.1
2. Coloque em `God of Create/mods/`
3. Reinicie o cliente

## 🐛 Suporte e Troubleshooting

### Problemas Comuns

**Jogo não inicia?**
- Verifique se tem 12+ GB de RAM alocada
- Confirme a versão do Java (17+)
- Desative mods problemáticos em `mods/`

**Crashes na inicialização?**
- Consulte `crash_assistant_localization_overrides/README.md`
- Verifique compatibilidade de versões
- Aumente RAM alocada

**Performance ruim?**
- Reduza distância de renderização (F3+G)
- Desative recursos visuais em Video Settings
- Feche aplicações de fundo
